# 验收与运维（V1）

## 1. 上线验收标准

必须全部满足：

- 每日 08:00 自动更新稳定运行
- 推荐接口可正常返回结果（Top4 或不足 4）
- 登录后可评分与评论
- 模型评分在 10 条后展示公开均分

## 2. 推荐质量验收

- 候选过滤规则生效：`has_aa && has_or`
- 严格模式生效：核心字段缺失模型被跳过
- 解释输出完整：每项至少 3 条理由 + 1 条 tradeoff

## 3. 稳定性与性能指标

- 数据任务成功率（按日）
- 有效候选数量（按日）
- 推荐接口 P95 耗时
- 推荐空结果率
- 评分提交成功率

## 4. 告警策略

- 数据任务失败立即告警
- 当日候选数较前一日下降超阈值告警
- 推荐接口错误率超阈值告警

## 5. 运行手册（建议）

- 任务失败排查顺序：抓取 -> 合并 -> 字段校验 -> 发布
- 服务异常排查顺序：DB 可用性 -> API 日志 -> 限流与鉴权
