## FAQ（4 条）

1. 这是 OpenRouter 替代品吗？  
不是。当前是决策层（推荐与评估计划），不是统一代调用执行层。

2. 我必须传候选模型列表吗？  
不必须。默认可从 Supabase `model_snapshots` 自动加载候选。

3. `SUPABASE_SERVICE_ROLE_KEY` 会暴露吗？  
不会。仅用于服务端环境变量，禁止放前端。

4. Skill 安装后会自动调用 API 吗？  
取决于你的 Agent 配置。Skill 定义流程与输出规范，是否联网调用由 Agent 运行环境决定。
