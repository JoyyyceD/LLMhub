# 用户流程（V1）

## 1. 推荐主流程

1. 用户进入推荐页
2. 选择主场景模板（如代码助手、科学推理等）
3. 选择偏好项（地区、预算、速度）
4. 可选使用高级筛选（上下文、输入类型、标签）
5. 提交请求并查看 Top4 推荐结果
6. 查看每个模型的关键指标与推荐理由

## 2. 社区评分帖流程

1. 用户登录
2. 进入模型详情页或社区发帖入口
3. 若在社区入口发帖，需先选择模型
4. 提交总评分（1-5，必填）
5. 可选填写维度评分、服务商、优缺点、评论
6. 系统校验“每用户每模型仅一条评分帖”
7. 重复提交走更新语义，发布后即时展示
8. 用户可对评分帖回复
9. 用户可对评分帖点赞或踩

## 3. 展示规则

- 当某模型评分数 `< 10`：不展示公开均分
- 当某模型评分数 `>= 10`：展示公开均分与评分量
- 评分帖列表始终可展示
- 社区列表默认按 `updated_at` 倒序
- 支持按模型筛选

## 4. 异常与边界流程

- 候选模型不足 4：返回实际可推荐数量，不降级凑数
- 用户未登录：允许浏览推荐，不允许评分与评论
- 当日数据异常：回退到上一次有效快照（由平台侧处理）
