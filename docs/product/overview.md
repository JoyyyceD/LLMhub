# 产品概览（V1）

## 1. 产品目标

LLMArena V1 提供三类核心能力：

- 模型推荐：用户输入需求后获得 Top4 推荐
- 评分系统：用户对模型进行 1-5 星评分
- 评论讨论：用户在模型下进行评论交流

## 2. 目标用户

- 直接面向 C 端用户
- 关注“该选哪个模型更合适”的个人与小团队

## 3. 核心价值

- 降低选型成本：用结构化偏好替代手工对比
- 提升可解释性：推荐结果给出理由与取舍
- 累积社区认知：评分与评论帮助新用户快速判断

## 4. V1 范围

- 推荐粒度：模型版本（`aa_slug`）
- 推荐数量：Top4（不足时允许少于 4）
- 输入方式：模板化选择 + 结构化偏好
- 数据更新：每日自动化刷新
- 社区能力：评分 + 评论（非论坛发帖）

## 5. 非目标（V1 不做）

- 不做 provider endpoint 粒度推荐
- 不做社区分数入模
- 不做复杂个性化推荐
