# 场景信息架构（V1）

## 1. 结构说明

- 一级场景共 8 个
- 每个一级场景包含：
- 1 个默认任务
- 5 个细分任务
- 形成 `8 x 6` 的信息架构

## 2. 一级与二级场景定义

### 2.1 通用对话（默认：日常问答）

细分：

1. 深度推理与决策
2. 规划与拆解
3. 信息抽取与结构化
4. 学习辅导与纠错
5. 对话模拟（面试/谈判/客服）

### 2.2 代码助手（默认：代码生成）

细分：

1. Bug 纠错
2. 代码重构
3. 代码解释/走读
4. SQL 查询与数据建模
5. 单元测试生成

### 2.3 科学推理（默认：科学问题解释与推断）

细分：

1. 科研阅读与总结（论文/报告）
2. 实验设计与因果推断
3. 数据分析与统计推理（A/B、回归直觉）
4. 工程估算与数量级推理（Fermi、单位检查）
5. 科学计算/公式推导（推导 + 验证）

### 2.4 文案创作（默认：营销/增长文案）

细分：

1. 品牌定位与口吻（Brand voice）
2. 社媒短内容（短帖/脚本/标题）
3. 长文写作（博客/白皮书）
4. 改写润色（清晰/专业/口语）
5. 多版本 A/B（多风格/多受众）

### 2.5 长文档 RAG（默认：文档问答）

细分：

1. 长文总结（Executive summary）
2. 信息抽取（字段/实体/表格化）
3. 跨文档对比（差异/冲突点）
4. 证据引用与可追溯（出处链路）
5. 低幻觉模式（不确定就说不确定）

### 2.6 智能体 Agent（默认：任务执行）

细分：

1. 工具调用/函数调用（API/DB）
2. 工作流编排（多步/条件分支/幂等）
3. 多智能体协作（分工/互审）
4. 记忆与偏好（长期上下文）
5. 安全与权限（最小权限/脱敏/审计）

### 2.7 多模态（默认：图像理解）

细分：

1. OCR/表单抽取（票据/合同页）
2. 图表理解（图形/仪表盘）
3. UI/截图诊断（产品/报错截图）
4. 视觉推理（空间/步骤判断）
5. 图文混合材料问答（多模态 RAG）

### 2.8 翻译（默认：通用翻译）

细分：

1. 专业领域翻译（法/医/金/技）
2. 本地化（口吻/文化适配）
3. 术语一致性（术语表/统一表达）
4. 校对润色（纠错 + 风格）
5. 低延迟短句（实时/同声偏好）

## 3. 编码建议

- API 建议同时接收：
- `scenario`：一级场景
- `sub_scenario`：二级场景（可选，未传则走默认任务）
- 配置层按 `scenario + sub_scenario` 决定：
- 质量公式
- 模板权重微调
- 解释模板
